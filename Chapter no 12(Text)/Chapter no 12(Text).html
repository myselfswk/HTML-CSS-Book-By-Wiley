Specifying Typefaces

font-family

The font-family property allows you to specify the typeface that should be used for any text inside the element(s) to which a CSS rule applies. The value of this property is the name of the typeface you want to use. The people who are visiting your site need the typeface you have specified installed on their computer in order for it to be displayed.

font-size

The font-size property enables you to specify a size for the font. There are several ways to specify the size of a font. The most common are:

Note:- (Text Size is smaller because of css file have font-size attribute '12px')

  • Pixels:

    • Pixels are commonly used because they allow web designers very precise control over how much space their text takes up. The number of pixels is followed by the letters px.

  • Percentages:

    • The default size of text in browsers is 16px. So a size of 75% would be the equivalent of 12px, and 200% would be 32px.

@font-face

@font-face allows you to use a font, even if it is not installed on the computer of the person browsing, by allowing you to specify a path to a copy of the font, which will be downloaded if it is not on the user's machine.
