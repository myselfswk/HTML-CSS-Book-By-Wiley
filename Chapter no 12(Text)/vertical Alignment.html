Alignment

text-align

The text-align property allows you to control the alignment of text. The property can take one of four values:

  1. left

    2. This indicates that the text should be left-aligned.

  2. right

    3. This indicates that the text should be right-aligned.

  3. center

    4. This allows you to center text.

  4. justify

    5. This indicates that every line in a paragraph, except the last line, should be set to take up the full width of the containing box.

Vertical Alignment

vertical-align

The vertical-align property is a common source of confusion. It is not intended to allow you to vertically align text in the middle of block level elements such as p and div element, although it does have this effect when used with table cells (the td and th elements).

It is more commonly used with inline elements such as img, em, or strong elements. When used with these elements, it performs a task very similar to the HTML align attribute used on the img element
